### OSI layer system 
We have OSI layer as a 7 Layer system 
Application Layer 
Presentation layer 
Session Layer 
Transport Layer 
Network Layer 
Data Link Layer 
Physical Layer 

#### Application Layer  
- > Peer to peer Protocols 
- > Client Server Protocol
